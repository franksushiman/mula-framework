# Definição de Nó — MULA V3

Um nó é uma entidade soberana que opera o MULA localmente.

## Um nó pode ser:
- uma pessoa
- uma família
- uma comunidade
- uma organização
- um coletivo

## Propriedades fundamentais
- Opera seu próprio canal de comunicação (ex: WhatsApp)
- Define sua frota interna
- Mantém soberania local
- Não controla a rede
- Não pode bloquear cooperação externa

## Cooperação
- Se não houver frota interna disponível,
  a busca externa é obrigatória.
- A rede existe pela necessidade, não por permissão.

Este documento é público, auditável e inegociável.
