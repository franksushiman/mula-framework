# Instância Padrão — MULA V3

Toda instância do MULA V3 parte da mesma configuração base.

## O que é igual em todas as MULAs
- Core e guards
- Compartilhamento obrigatório de frota
- Contratos públicos
- Regras de soberania

## O que muda por instância
- Nome do nó
- Canal de comunicação
- Frota local
- Interface (site, WhatsApp ou ambos)

Não existem instâncias privilegiadas.
