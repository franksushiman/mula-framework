# Contrato Econômico — MULA V3

O MULA é um framework comunitário.
Ele não é marketplace, não intermedia pagamentos e não define preços.

## Princípios

- Monetização é local e opcional
- O valor pertence ao nó que fornece a frota
- Nós-ponte não capturam valor
- O compartilhamento de frota é estrutural e obrigatório

## Observação

Este contrato é público, auditável e inegociável.
